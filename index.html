MacOs系统中python的安装

Posted on 2020-10-26 In note

MacOs里python自带easy_install的，

terminal(终端)：sudo easy_install pip (sudo 管理员权限)

运行完可以用pip help测试一下是否安装成功，

terminal(终端)：pip help

成功安装后，直接‘ pip install numpy ’或者其他包就可以了。

但是最好不要在Mac自带的python里面折腾，因为一旦折腾崩了可能影响系统。

建议安装Homebrew，然后brew install python，也可以指定版本安装。然后将这个python的环境变量设置在系统自带的前面即可。这样一来有几个好处：
1.随意折腾，挂了系统不会受影响；
2.这种方式安装的自带pip；
